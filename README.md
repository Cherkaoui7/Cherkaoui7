## 🏗️ Architecture Patterns

### 1. **MERN Stack**
```
React (UI) ↔ HTTP/JSON ↔ Express.js (API) ↔ Mongoose ↔ MongoDB
```

### 2. **Real-Time Architecture**
```
React Client ↔ WebSocket (Pusher/Echo) ↔ Express ↔ MongoDB
                                  ↓
                         Event Broadcasting
```

### 3. **Monorepo (Turborepo)**
```
Root (car_shop/)
├── packages/
│   ├── @carshop/schema (Shared types)
│   ├── @carshop/db (DB utilities)
│   └── @carshop/api-client (SDK)
└── apps/
    ├── web (React)
    ├── api (Express)
    ├── mobile (React Native)
    └── android (Kotlin)
```

### 4. **Multi-Platform**
```
Shared Types & Business Logic
        ↓
    ┌───┼───┬───┬────┐
    ▼   ▼   ▼   ▼    ▼
  Web API Mobile Android
    ↓   ↓   ↓   ↓    ↓
         MongoDB Database
```

### 5. **Python AI Pipeline**
```
FastAPI Server (Uvicorn)
    ├─ HTML→PDF (ReportLab)
    ├─ Background Removal (rembg)
    ├─ Video Processing (moviepy)
    └─ Screenshots (Playwright)
         ↓
    Data Processing (Numpy, Pillow)
```
